## Quickstart

Get up and running in under a minute.

<Tabs>
  <TabItem label="macOS">

```bash
# Install via Homebrew
brew install floruntime/tap/flo

# Start Flo
flo server start

# Store a value
flo kv set hello "world"

# Read it back
flo kv get hello
```

  </TabItem>
  <TabItem label="Linux">

```bash
# Download and install
curl -fsSL https://get.floruntime.io | sh

# Start Flo
flo server start

# Create a stream
flo stream create events

# Append an event
flo stream append events '{"type":"login","user":"alice"}'
```

  </TabItem>
  <TabItem label="Docker">

```bash
# Pull and run
docker run -p 4222:4222 -p 9090:9090 floruntime/flo:latest

# Use the CLI from another terminal
flo kv set mykey "hello from docker"
flo kv get mykey
```

  </TabItem>
</Tabs>
